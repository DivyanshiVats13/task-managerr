# Daily Flow – Task Manager

A simple browser-based task manager. No frameworks, no backend — just HTML, CSS, and vanilla JavaScript.

## What it does

- Add tasks and mark them complete
- Delete individual tasks or clear the whole board
- Shows a greeting based on time of day (morning / afternoon / evening)
- Tracks and displays your progress count
- Saves tasks to localStorage so they persist on refresh

## Tech used

- HTML, CSS, JavaScript (vanilla)
- [Phosphor Icons](https://phosphoricons.com/) for UI icons
- localStorage for data persistence

## How to run

Just open `index.html` in any browser. No installs, no build step.
